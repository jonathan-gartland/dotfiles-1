
# Introduction
    Instructions for setting-up dot-files-forest.

# Setup

0) Install python, if necessary
    
1) Download install.py

2) Clone Repos

    ```
    cd $HOME
    ~/dot-files-forest/install.py -c
    ```

3) Create Links

    ```
    cd $HOME
    ~/dot-files-forest/install.py -l home|work|none
    ```
